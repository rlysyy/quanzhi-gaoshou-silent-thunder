# 第十八章 外界的眼睛

就在林黯在训练室里与"不确定性"搏斗时，外界对她的讨论已经愈演愈烈。

《电竞之家》用整整两个版面分析了她的首轮表现：

**"数据天才还是战术隐患？——蓝雨新人林黯的双面性"**

文章开篇肯定了林黯的操作天赋，用大量数据证明她的手速和精度在联盟历史上都排得上号。但话锋一转，笔者的质疑随之而来：

"然而，当我们深入分析林黯的比赛录像，会发现一个令人担忧的现象：她的所有决策都基于预设模型。在与三零一度的比赛中，杨聪仅用了一个简单的诱饵战术，就让她陷入了长达三十秒的迷茫期。如果不是黄少天的及时支援，比赛结果可能完全不同。

"荣耀是人与人的对抗，不是人与数据的对抗。当对手开始研究林黯的模型，当她的每一个选择都被预判，这位天才少女还能保持现在的惊艳表现吗？

"蓝雨战队将如此重要的战术位置交给一个十四岁的新人，是天才的赌博，还是危险的冒进？时间会给我们答案。"

文章在荣耀圈引起了轩然大波。支持者和反对者在论坛上吵翻了天。

"小编懂个屁！林黯的操作肉眼可见的强，模型怎么了？能赢就是好模型！"

"但小编说得有道理啊，电竞历史上多少数据流选手被研究透就废了？林黯还太年轻，蓝雨这是在拔苗助长。"

"十四岁打职业本来就不靠谱，心理承受能力、战术素养、团队配合，这些都是需要时间积累的。蓝雨为了噱头毁了天才，可悲。"

"楼上酸什么酸？有本事让你主队也找个十四岁的天才啊！"

各种言论甚嚣尘上，而真正的职业圈，则用行动表达了他们的态度。

常规赛第二轮，蓝雨客场对阵轻裁战队。

赛前发布会，轻裁战队的队长，联盟著名的"数据控"选手诸葛明，在记者提问时推了推眼镜，慢条斯理地说：

"林黯选手的打法很有意思，是典型的数据驱动型。我们队的数据分析师已经建立了她的行为模型，预测准确率目前在73%左右。今天的比赛，我们会验证这个模型的可靠性。"

这话说得客气，但背后的意思很明显：我们研究透你了。

更麻烦的是，轻裁战队的战术风格恰好克制林黯。

他们的核心阵容是"四保一"——四个辅助型角色保护一个核心输出。这个核心不是强攻手，而是陷阱专家。

盗贼职业，专精陷阱，地图控制能力极强。

"麻烦了。"去赛场的车上，黄少天难得正经，"打轻裁最烦的就是满地图的陷阱，走一步炸一下。小黯你是刺客，需要机动性，但他们的陷阱专治机动性。"

"而且诸葛明那家伙，是真的会用数据阴人。"徐景熙补充，"上赛季我们打轻裁，队长被他用数据模型预判了三次关键技能，输得憋屈。"

林黯看着平板上轻裁战队的比赛录像，手指在诸葛明的数据统计上停留。

"他的手速峰值只有291，联盟中下游。但战术决策准确率87%，联盟第一。"她低声说，"他是用脑子打游戏的类型。"

"而且是专门研究别人脑子的类型。"黄少天哼了一声，"小黯，这场你的压力会很大。诸葛明肯定针对你设计了一整套陷阱套餐。"

"我知道。"林黯关闭平板，看向窗外飞逝的街景，"但我也有准备。"

"哦？什么准备？"

"秘密。"

黄少天瞪大眼睛："好哇小丫头，学会卖关子了！快说快说！"

林黯没理他，闭上眼睛开始养神。但她的手指在膝盖上轻轻敲击，那是在模拟操作的热身动作。

比赛场馆很快就到。

轻裁战队的主场在B市，一座老旧但氛围狂热的电竞馆。观众席上，轻裁的绿色队旗几乎铺满整个看台。但在东北角，有一小片蓝色倔强地存在着——那是随队出征的蓝雨粉丝，以及……专程来看林黯的人。

"林黯！加油！"有年轻女孩的喊声从看台传来。

林黯抬头看去，看到几个女孩举着"暗影无声"的手绘海报，在绿色的海洋中格外显眼。她愣了愣，然后很轻地点了点头。

这个细微的互动被摄像头捕捉，投放在大屏幕上，引起一片尖叫。

"看来林黯选手已经有粉丝了。"解说台上，潘林笑道。

"有实力的人自然会有粉丝。"李艺博说，"但今天她要面对的是诸葛明的数据陷阱。让我们看看，数据天才和数据控，谁会笑到最后。"

个人赛，蓝雨两胜一负，暂时领先。

关键的团队赛即将开始。

地图载入：机关城。

看到这张图，蓝雨众人的脸色都变了。

"我去！诸葛明你这老阴比！"黄少天在团队频道骂出声。

机关城，轻裁战队的王牌地图。整张图就是一个巨大的机关迷宫，墙壁会移动，地板会翻转，暗箭、落石、陷阱无处不在。更恶心的是，这些机关的触发机制是随机的，每次比赛都不一样。

"这张图没法计算。"喻文州的声音依然平静，但语速快了一些，"机关触发没有规律，只能靠反应。小黯，你的模型在这里基本没用。"

"我知道。"林黯看着载入界面，深吸一口气，"所以这场，我不准备用模型。"

"那你用什么？"

"用你们教我的东西。"

倒计时结束。

比赛开始。

蓝雨出生在机关城的东区，轻裁在西区。按照常规战术，双方会向中央区域移动，在迷宫中心交战。

但轻裁没有动。

"他们在等我们过去。"喻文州判断，"诸葛明肯定在必经之路上布置了陷阱。少天，你走左路探路。小卢右路。小黯……"

"我走中路。"林黯说。

"不行，太危险。"

"中路最危险，所以他们不会想到我走中路。"林黯已经操控暗影无声开始移动，"而且，刺客有'陷阱感知'的被动技能，我能提前发现一部分陷阱。"

"那也……"

"让她去。"黄少天突然说，"队长，相信她。"

团队频道安静了两秒。

"……好。小黯，你走中路，但随时报告情况。一旦发现不对，立刻后撤。"

"明白。"

暗影无声如一道黑影，消失在迷宫复杂的通道中。

中路的通道很窄，两侧是高耸的石墙。墙上有各种奇怪的雕刻，有些是装饰，有些……可能是机关触发点。

林黯没有计算。她关闭了大脑中的数学模型，全凭感觉前进。

第一步，安全。
第二步，安全。
第三步，右脚落地的瞬间，她听到了极其轻微的"咔哒"声。

陷阱！

但身体比大脑更快。在听到声音的瞬间，她的手指已经按下后跳键。暗影无声向后翻滚，几乎同时，三支弩箭从墙壁的暗孔中射出，钉在她刚才站立的位置。

"中路有陷阱，弩箭，触发点在第三步的右侧墙壁，第三块石砖。"她在团队频道快速报告。

"收到。少天，左路情况？"

"暂时安全，但总觉得毛毛的……"

林黯继续前进。这一次，她走得更慢，更谨慎。每一步落下前，都会仔细观察地面和墙壁。

第五步，她停住了。

前方的地面上，有一道几乎看不见的裂缝。按照常规思路，这可能是翻板陷阱。但……

感觉不对。

她弯腰，捡起一块碎石，扔向裂缝前方——不是裂缝上，是裂缝前面一步的位置。

碎石落地，"轰"的一声，整个通道的顶部塌了下来。

是落石陷阱，但触发点不在裂缝，在裂缝前一步！

"阴险。"林黯低语，然后绕开陷阱区，继续前进。

她就用这种笨办法，一步一步地探路，一步一步地破解陷阱。速度很慢，但很稳。

五分钟后，她到达了中路通道的尽头——一个开阔的大厅。

大厅中央，轻裁战队的五人已经严阵以待。诸葛明的盗贼站在最前面，看到她出现，在公共频道打了个笑脸表情。

"比预计的慢了47秒，林黯选手。你的谨慎超出了我的模型预测。"

"你的陷阱也超出了我的模型预测。"林黯回应。

"那真是荣幸。不过，游戏到此为止了。"

话音落下，大厅四周的墙壁突然开始移动。八道石门同时落下，将整个大厅封闭成一个密室。

"瓮中捉鳖。"诸葛明打字，"欢迎来到，我的主场。"
